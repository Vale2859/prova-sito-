# Farmacia Montesano - versione statica pronta per GitHub Pages

## Come pubblicare
1. Carica tutti i file nella root del repository GitHub.
2. Attiva GitHub Pages dal branch principale.
3. Assicurati che `index.html` sia nella cartella principale.

## Note
- Il progetto è una ricostruzione statica ispirata al sito pubblico.
- Funziona senza build, plugin o framework.
- Puoi personalizzare testi, immagini e collegamenti pagina per pagina.
